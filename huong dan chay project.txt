Hướng dẫn chạy project
- Front-End
- framework sử dụng là vuejs
- cài đặt nodejs để tạo môi trường hoạt động phiên bản đang dùng là bản node 18.16
- cài đặt vue hiện tại trong project là đang chạy bản vue 3 có thể lên trang chủ để tham khảo https://cli.vuejs.org/guide/installation.html
- câu lệnh cài đặt: npm i , npm run dev
-Back-End
- công nghệ sử dụng php và laravel
- cài đặt xampp để tạo môi trường hoạt động phiên bản đang dùng là bản xamppp 8.2.4
- cài đặt composer để tạo môi trường hoạt động
- phiên bản laravel hiện tại đang là laravel 9
- tạo database trên mysql của xampp để có db để hoạt động
- câu lệnh cài đặt:
- composer i
- php artisan migrate
- php artisan serve
